# 문제
* 재귀를 활용하여 최대공약수를 계산하시오.  
* 유클리드 호제법   
  ```
  a = 10, b = 12
  10 % 12 = 10
  12 % 10 = 2
  10 % 2 = 0
  ```  
   
# 입출력 조건
* 입력은 한 줄로 정수 $a$ 와 $b$ 두 개가 주어지며 한 칸의 공백으로 구분한다.
* $1 \lt a,\ b \lt 10,000$
   
## 입출력 예시
### 입력 예시
```
19332 78696
```
### 출력 예시
```
36
```
#### 예시 설명
```
78696 ＝ 19332 × 4 ＋ 1368 
19332 ＝ 1368 × 14 ＋ 180 
1368 ＝ 180 × 7 ＋ 108 
180 ＝ 108 × 1 + 72 
108 ＝ 72 × 1 ＋ 36
72 ＝ 36 × 2
```
