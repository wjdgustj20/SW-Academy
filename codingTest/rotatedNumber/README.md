# 문제
* 문제에서 주어지는 자리 수(n)에 해당하는, 회전해도 동일한 숫자를 모두 찾고 그 값들의 개수를 출력하시오.   
* 회전해도 동일한 숫자 예시  
  ```
  a = 10, b = 12
  10 % 12 = 10
  12 % 10 = 2
  10 % 2 = 0
  ```  
   
# 입출력 조건
* 입력은 한 줄로 정수 a와 b 두 개가 주어지며 한 칸의 공백으로 구분한다.
* $1 \lt a,\ b \lt 10,000$
   
## 입출력 예시
### 입력 예시
```
19332 78696
```
### 출력 예시
```
36
```
#### 예시 설명
```
78696 ＝ 19332 × 4 ＋ 1368 
19332 ＝ 1368 × 14 ＋ 180 
1368 ＝ 180 × 7 ＋ 108 
180 ＝ 108 × 1 + 72 
108 ＝ 72 × 1 ＋ 36
72 ＝ 36 × 2
```
