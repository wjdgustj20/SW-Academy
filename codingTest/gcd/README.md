# 문제
* 재귀를 활용하여 최대공약수를 계산하시오.  
* 유클리드 호제법   
  * $a=10,\ b = 12$   
    $10 \% 12 == 10$   
    $12 \% 10 == 2$   
    $10 \% 2 == 0$   
   
# 입출력 조건
* 입력은 정수 $N$ 한 개가 주어진다.
* $2 \le N \le 30$
   
## 입출력 예시
### 입력 예시
```
7
```
### 출력 예시
```
13
```
#### 예시 설명
* 피보나치 수열: $0,\ 1,\ 1,\ 2,\ 3,\ 5,\ 8,\ 13$
* $F_7 = 13$
